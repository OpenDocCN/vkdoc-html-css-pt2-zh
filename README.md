## 下载

### Docker

```
docker pull apachecn0/flygon-html-css-zh-pt3
docker run -tid -p <port>:80 apachecn0/flygon-html-css-zh-pt3
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flygon-html-css-zh-pt3
flygon-html-css-zh-pt3 <port>
# 访问 http://localhost:{port} 查看文档
```
